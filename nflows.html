Normalizing flows for multivariate model calibration

Method description

Let’s say \(y_0 \in \{0,1\}\) with \(y_0=1\) indicating the observation is from a weather station and \(y_0=0\) indicating it is from a climate model. There are two variables we are interested in are \(y_1\) is precipitation and \(y_2\) is temperature. Together the data are \(y = (y_0,y_1,y_2)\).

Let’s work off of this normalizing flows framework. I started here because it ends up being effectively quantile mapping which we understand and we can use SPQR which we said we would do in the proposal. In their notation, the latent variable is \(z\) and follows the standard normal distribution and the transformation are \(f\). Below we use \(u\) for the latent variable with \(u\in[0,1]\) for the quantile level and \(f\) is called \(q\) because it is a quantile function.

The model has latent variable \(u = (u_0,u_1,u_2)\) relates to the data as \(y_0=u_0\), $$y_1 = q_1(u_1; y_0) \mbox{ and } y_2 = q_2(u_2; y_0,y_1).$$ The functions \(q_1\) and \(q_2\) are conditional quantile functions that we learn using deep quantile regression via SPQR. Note that the conditioanl quantile function depend on \(y_0\) and so we model a different distribution for monitors and model output.

After we learn the conditional quantile functions we can bias correct. For a given observation from the model output with \(y_0=u_0=0\) we compute $$u_1^* = q_1^{-1}(y_1; y_0) \mbox{ and } u_2^* = q_2^{-1}(y_2; y_0,y_1).$$ Then the corrected \(y_1\) and \(y_2\) are computed by setting \(u_0^*=y_0^*=1\) and simply running \(u^* = (u_0^*,u_1^*,u_2^*)\) through the quantile functions $$y_1^* = q_1(u_1^*; y_0^*) \mbox{ and } y_2^* = q_2(u_2^*; y_0^*,y_1^*).$$ These corrected values should have the same joint distribution as the monitoring data.

Define the quantile functions and their inverses

In this silly example we pretend we know the conditional quantile functions exactly and so the calibration is perfect. In reality, we would have to estimate the conditional quantile function using SPQR trained on both monitor data and model output.

# Conditional quantile function for precip
q1 <- function(u1,y0){  
   a <- 3+0.5*y0
   b <- 4-1.0*y0
   y <- qgamma(u1,a,b)
return(y)}

# Conditional inverse quantile function for precip
p1 <- function(y1,y0){
   a <- 3+0.5*y0
   b <- 4-1.0*y0
   u <- pgamma(y1,a,b)
return(u)}

# Conditional quantile function for temperature
q2 <- function(u2,y0,y1){
   m <- 1.0-0.5*y1 + 0.5*y0*log(y1)
   s <- 0.2*y1*(1-y0)+.2
   y <- qnorm(u2,m,s)
return(y)}

# Conditional inverse quantile function for temperature
p2 <- function(y2,y0,y1){
   m <- 1.0-0.5*y1 + 0.5*y0*log(y1)
   s <- 0.2*y1*(1-y0)+.2
   u <- pnorm(y2,m,s)
return(u)}

Generate data

set.seed(919)
n  <- 100000
y0 <- rbinom(n,1,.8)
u1 <- runif(n)
y1 <- q1(u1,y0)
u2 <- runif(n)
y2 <- q2(u2,y0,y1)

Bias correction

 U1 <- p1(y1,y0)     # solve for the latent factors
 U2 <- p2(y2,y0,y1)
 Y0 <- rep(1,n)      # set everything to "observations"
 Y1 <- q1(U1,Y0)     # Project from latent to observed space
 Y2 <- q2(U2,Y0,Y1)

 plot(y1,Y1,xlab="Original",ylab="Corrected",main="Precip",col=y0+1)
 abline(0,1)
 legend("topleft",c("Model","Observations"),col=1:2,lwd=2,bty="n")


 plot(y2,Y2,xlab="Original",ylab="Corrected",main="Temp",col=y0+1)
 abline(0,1)
 legend("topleft",c("Model","Observations"),col=1:2,lwd=2,bty="n")
